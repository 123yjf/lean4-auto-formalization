-- 3. [Proof] 
--    - Algebraic identity: For any x, y, z, (3x + y) + (5y + z) + (3x + 5z) = 6x + 6y + 6z (by expanding and collecting terms).
--    - Numeric sum: 17 + 14 + 41 = 72.
--    - Therefore, any x, y, z satisfying the first three equations also satisfy 6x + 6y + 6z = 72. Since 6x + 6y + 6z = 6(x + y + z), it follows that 6(x + y + z) = 72, hence x + y + z = 12.

--    - For this triple, the sum is x + y + z = 66/13 + 23/13 + 67/13 = 156/13 = 12.
--    - Hence this explicit triple satisfies all four required equations.
